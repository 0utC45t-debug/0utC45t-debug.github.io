一、Docker Desktop 安装

📋 环境信息

组件 版本
Windows 11 Insider Preview 26200.8457 (Canary)
WSL Ubuntu 24.04.3 LTS (WSL2)
Docker Desktop 4.43.2
Docker Engine 28.3.2

⚠️ 遇到的问题

1. 安装时 DISM 报错

原因：预览版系统移除了旧版功能名称，安装程序尝试启用 Microsoft-Hyper-V 失败。
解决：忽略该错误，核心组件已安装完成。

2. 桌面图标点击无反应 / 启动慢

原因：Electron 前端与预览版系统兼容性差，静默崩溃。
解决：

  • ✅ 使用系统托盘 🐳 图标管理
  • ✅ 使用网页版 localhost:8888（秒开）
  • ✅ 命令行操作（推荐）

✅ 验证成功
