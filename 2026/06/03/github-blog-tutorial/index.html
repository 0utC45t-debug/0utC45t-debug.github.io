3.使用github pages

(1) 配置 SSH key

目的：实现免密上传到 GitHub，无需每次都输入账号密码

检查本地是否已有 SSH key：

如果提示路径不存在，说明还没配置过，直接进行下一步。

(2) 生成 SSH key：

连续按三次回车即可（使用默认设置，不设置密码）。

生成成功后，会在 C:\Users\你的用户名\.ssh 目录下生成两个文件：

  • id_rsa（私钥，千万不要泄露）
  • id_rsa.pub（公钥，需要上传到 GitHub）

(3) 获取公钥内容：

或者手动进入 .ssh 文件夹，用记事本打开 id_rsa.pub，复制全部内容（从 ssh-rsa 开头到邮箱结尾，不要有多余空格或换行）。

添加到 GitHub：

  1. 登录 GitHub，点击右上角头像 → Settings
  2. 左侧菜单选择 SSH and GPG keys
  3. 点击绿色按钮 New SSH key
  4. Title 随便填（如 MyBlog），Key 粘贴刚才复制的公钥内容
  5. 点击 Add SSH key（可能需要输入 GitHub 密码确认）

测试连接：

第一次连接会提示确认，输入 yes 回车。如果看到类似这样的提示，说明配置成功：

(4) 创建 GitHub 仓库

  1. 登录 GitHub，点击右上角 + → New repository
  2. Repository name（仓库名）：必须填写 你的用户名.github.io,否则没有用喵。这个仓库名将作为你博客的访问地址
  3. 保持 Public（公开）
  4. 不要勾选 “Initialize this repository with a README”（保持空白）
  5. 点击 Create repository

5. 常用命令速查

命令 说明
hexo new "文章名" 新建文章
hexo clean 清除缓存
hexo generate (或 hexo g) 生成静态网页
hexo server (或 hexo s) 本地预览（http://localhost:4000）
hexo deploy (或 hexo d) 部署到 GitHub
hexo version 查看 Hexo 版本
