01.Markdown语法

|浏览量:

一份详细、结构清晰的 Markdown 语法笔记，适合快速查询与学习。
保持简洁，专注内容，让排版自然生效。

一、基础语法

1. 标题（Headings）

使用 # 表示标题级别，从一级（最大）到六级（最小）。

2. 文本样式（Text Styles）

  • 粗体：用 ** 或 __ 包裹文本
    **这是粗体** 或 __这也是粗体__

  • 斜体：用 * 或 _ 包裹文本
    *这是斜体* 或 _这也是斜体_

  • 粗斜体：用 *** 或 ___ 包裹
    ***粗斜体文本***

  • 删除线：用 ~~ 包裹
    ~~删除线文本~~

  • 高亮：部分渲染器支持，用 == 包裹
    ==高亮文本==

四、使用建议与技巧

  1. 兼容性：不同平台（GitHub、Typora、VS Code 等）对 Markdown 的支持可能略有差异，建议先测试。
  2. 转义字符：使用 \ 来显示原本有特殊意义的字符，如 \* 显示为 *。
  3. 保持简洁：Markdown 的设计哲学是易于读写，避免过度使用 HTML 或复杂结构。
  4. 工具推荐：
    • 编辑器：Typora、VS Code、Obsidian
    • 在线工具：Markdown Here、StackEdit

五、总结

Markdown 是一种轻量级标记语言，通过简单的符号实现优雅的排版。掌握基础语法后，可以应对大多数写作场景，再根据需要学习扩展语法。
